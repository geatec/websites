Fundamental class C++

Verzorgd door: ir. Jacques de Hooge

 

Voor wie is deze fundamental class bestemd

 

Je hebt enige ervaring met programmeren voor je beroep of gewoon voor je plezier.  Ook heb je belangstelling voor technische,  reken- of simulatie/game toepassingen.  Je weet dat C++ hiervoor vrijwel onbegrensde mogelijkheden biedt, maar je weet ook dat het geen gemakkelijke programmeertaal is. Je bent van plan je deze taal grondig eigen te maken. Naast het volgen van lessen ben je bereid hier ook buiten lestijd de nodige energie en tijd in te steken. Je gaat niet voor het minimum, maar wilt hier werkelijk goed in worden.

 

Wat leer je

 

Na afloop van de fundamental class beheers je op professioneel niveau gebruik van de volgende taalfaciliteiten:

 

·      Het hiërarchisch, systematisch gebruik van modules en het vastleggen van module interfaces door middel van header files.

·      Basis gegevenstypen, pointers, references en arrays.

·      Functies met value, reference en pointer parameters en returns en overloading.

·      Dynamisch geheugenbeheer met new en delete.

·      Objectoriëntatie: inkapseling, access classes, meervoudige overerving, polymorphism, virtual functions.

·      Juist gebruik van diverse constructor en destructor typen en constructor overloading.

·      Overloading van de *, /, +, -, <<, >>, [] en () operators.

·      Function templates en class templates.

·      De belangrijkste faciliteiten van de Standard Template Library.

 

Hoe zit de fundamental class C++ in elkaar

 

De fundamental class C++  volg je in een groep van maximaal 10 personen. De kosten per les van 2 uur bedragen €40, -. Je ontvangt een factuur. Het aantal lessen hangt af van je voorkennisniveau. Met enige programmeerervaring, andere taal geen bewaar, zul je meestal rond de 10 lessen nodig hebben om je de boven beschreven kennis eigen te maken.

 

Voor aanvang van je fundamental class C++ heb je een startgesprek. Pas als we naar aanleiding daarvan beiden besluiten dat we met elkaar willen werken, ga je een leerovereenkomst aan.

 

Succesvolle afronding van de fundamental class houdt in dat jij je het opgesomde onder het kopje 'Wat leer je' hebt eigengemaakt.  Dit resulteert in een door mij ondertekend certificaat en mijn bereidheid je bij sollicitaties aan te bevelen aan de hand van een realistische schets van je capaciteiten.

 

Praktische zaken

 

Indien je na het startgesprek de fundamental class C++ wenst te volgen, sluit je de leerovereenkomst rechtstreeks met mij. Het lesgeld dient na het startgesprek, maar voor de eerste reguliere les te worden voldaan. Omdat op individueel niveau wordt lesgegeven, wordt ook de startdatum individueel afgesproken. De cursisten bevinden zich dus in verschillende 'fasen'.

 

Je bent zelf verantwoordelijk voor het meebrengen van een eigen laptop met daarop geïnstalleerd een goede C++ compiler, bijvoorbeeld Microsoft, GCC of CLANG. Het is van belang dat dit alles goed werkt voor je met de fundamental class begint. De ervaring is dat je anders heel veel kostbare tijd verliest. Je werkt in de lessen maar ook thuis aan je software. Fabrikant-specifieke uitbreidingen en door wizzards gegenereerde code worden tijdens de fundamental class buiten beschouwing gelaten.

 

Wat daarnaast nodig is, is een goede code editor, bijvoorbeeld Notepad++ of GEdit. Hoewel source level debuggers en IDE's hun nut kunnen hebben en je ze zonder meer kunt gebruiken, wordt hieraan in de lessen geen eisen gesteld of aandacht besteed. Een goed mentaal model van je code en een command line compiler is alles wat je nodig hebt.

 

De cursuslocatie hang af van het aantal deelnemers. Gestreefd wordt naar een locatie binnen een straal van 5 km van het centrum van Rotterdam.

 

Tot slot

 

C++ is niet de makkelijkste taal om te leren. Het is echter het fundament van vrijwel elke huidige "technology stack". In sommige toepassingsgebieden, zoals besturingssystemen, compilers, gaming, simulaties, technisch wetenschappelijke berekeningen en real time controls is dat heel zichtbaar, maar ook indien je internet toepassingen laagje voor laagje gaat afgraven, kom je uiteindelijk C++ tegen, in de webserver, de interpreter van je scripttaal of in de code van je browser.

 

Tegelijkertijd is de grote massa van de ontwikkelaars zo gefocust op de bovenste, meest zichtbare van al deze laagjes, en daarmee op talen als JavaScript, dat bedrijven grote moeite hebben de nodige C++ ontwikkelaars in te huren. Voor hen een nadeel, voor jou een voordeel. De C++ wereld is dynamisch en het toneel van voortdurende innovatie.

 

Als mensen een huis uitzoeken, vragen ze zich meestal niet af hoe de heipalen eruit zien. Daar waar een fabrikant van modieuze gevelbekleding misschien over 10 jaar de bakens moet verzetten, is de toekomst van een funderingsbedrijf over het algemeen onomstreden. Naar mijn inzicht geldt dit ook voor C++.

 

Ik ben regelmatig betrokken geweest bij het inhuren van C++ programmeurs. Zo'n 70% ervan gaat glazig kijken als begrippen zoals heap management en pointer arithmetic worden aangesneden. Gaat het dan ook nog over constructor overloading en templates, dan stijgt dit percentage tot 90%. Ze vallen in de eerste ronde af. In de C++ fundamental class leer je die dingen die nodig zijn om ook echt op professioneel niveau als C++ programmeur aan de slag te kunnen.
